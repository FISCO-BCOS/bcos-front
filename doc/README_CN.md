# 两阶段并行共识

bcos-front是[FISCO BCOS 3.x](https://github.com/FISCO-BCOS/FISCO-BCOS)的区块链网络前置模块，区块链节点通过bcos-front与网关进行通信，并将消息转发到其他区块链节点。
